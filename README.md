### Using Git LFS:

 - To add file types to track: e.g. `git lfs track "*.so"` to track files of type .so
 - To convert already committed files to lfs: e.g. for file type .so `git lfs migrate import --include "*.so"`
 - Then commit/push as usual
 - For more info: https://git-lfs.com/
